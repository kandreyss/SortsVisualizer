# SortsVisualizer

Небольшой учебный проект на Java — визуализатор алгоритмов сортировки. Главная идея проекта — показать наглядно, как работают базовые алгоритмы сортировки, дать возможность увидеть поведение массивов при разных операциях (сравнениях, обменах) и почувствовать разницу в подходах.

Проект задуман как простой образовательный инструмент: код демонстрирует логику алгоритмов и их динамику, чтобы облегчить понимание для тех, кто только знакомится с сортировками. Это полезно для студентов и всех, кто хочет увидеть алгоритмы «в действии», а не только в виде теории.

Кому может пригодиться
- Тем, кто изучает алгоритмы и структуры данных.
- Тем, кто хочет понять различия между алгоритмами сортировки на интуитивном уровне.
- Для демонстраций на паре или при самостоятельной подготовке.

Кратко о содержимом
- Репозиторий содержит Java-код визуализатора сортировок.
- В коде реализована логика отображения процесса сортировки и основные механизмы для сравнения разных подходов.
- Это учебный, компактный проект — всё сделано так, чтобы было просто смотреть и разбирать.
